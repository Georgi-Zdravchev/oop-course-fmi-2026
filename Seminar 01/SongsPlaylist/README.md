## Задача - Плейлист 
_(решавана на семинар на 4та група, 17.02.2026)_

Въвеждат се N песни. За всяка песен се въвеждат:
- име (макс. 50 символа)
- изпълнител (макс. 50 символа)
- продължителност във формат `MM:SS` (приемаме, че няма песен над 1 час)
- категория(Pop, Rock, HipHop, Jazz, Latin, Other)

Програмата трябва да поддържа следните функционалности:

- извеждане на песните сортирани по продължителност (възходящо)
- филтриране на песните по изпълнител или по категория  
- филтър за песни с продължителност поне X секунди
- извеждане на общата продължителност на плейлиста, във формата `HH:MM:SS` (сума от времената)
- извеждане на “топ 3” най-дълги песни;
